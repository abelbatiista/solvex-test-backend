Run `pip install virtualenv`.

## Create Virtual ENV

Run `virtualenv env`.

## Create route specifying interpreter.

Run `virtualenv -p route/interpreter/python env`.

## Activate Virtual ENV

Run `env\Scripts\activate.bat`.
